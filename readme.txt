== Description ==
CZ: Zobrazuje na nástěnce nejnavštěvovanější články.
EN: Shows the most popular posts on the dashboard.

== Installation ==
CZ:
1. Uploadnout plugin na Váš blok, nebo nainstalovat přes správce pluginů.
2. Aktivovat.
3. Toď vše!
4. Plugin po spuštění hodinu získává data než je v tabulce zobrazí (návštěvy nejsou hned)
5. Od této chvíle plugin automaticky každou hodinu spouští aktualizaci tabulky

EN:
1. Upload the plugin to your blog or install via plugin management.
2. Activate it. 
3. You're done!
